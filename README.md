# celinette
celinette 반응형 웹

## 개발 배경
1. 본 앱은 대우정보처리회계학원 모바일포렌식을 활용한 응용SW개발자 양성과정 수업의 산출물로서, HTML, CSS, jQuery를 이용하여 제작한 웹 앱입니다.
1. Cordova로 패키징하고 github에서 형상관리를 하였습니다.
1. [Celinette](http://celinette.com/)를 참고 하였으며 반응형으로 리디자인 하였습니다.
